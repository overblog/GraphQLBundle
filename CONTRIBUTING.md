Contributing
============

Running tests
--------------

Install [phpunit](https://phpunit.de/manual/current/en/installation.html).

In the bundle directory:

```bash
bin/phpunit
```

Giving some love to PHP CS
---------------------------

```bash
composer require --dev 'friendsofphp/php-cs-fixer:^2.0'
bin/php-cs-fixer fix
```
